Q1:Write a query to find the addresses (location_id, street_address, city, state_province, country_name) of all the departments.

The answer is: SELECT * FROM `locations` INNER JOIN `countries`ON locations.country_id=countries.country_id;

Q2:Write a query to find the name (first_name, last name), department ID and name of all the employees.

The answer is: SELECT employees.first_name, employees.last_name, departments.department_id FROM `employees` INNER JOIN `departments`ON employees.department_id=departments.department_id;

Q3:Write a query to find the name (first_name, last_name), job, department ID and name of the employees who works in London.

The answer is:SELECT employees.first_name,employees.last_name, employees.job_id, `department_id` FROM `employees` WHERE country_id="LGW";

Q4:Write a SQL statement to rename the table countries to country_new.

The answer is:ALTER TABLE countries RENAME country_new;

Q5:Write a SQL statement to add a column region_id after state_province to the table locations.

The answer is:ALTER TABLE `locations` CHANGE `region_id` `region_id` INT(11) NOT NULL AFTER `state_province`;

Q6:Write a SQL statement to add a primary key for the columns location_id in the locations table.

The answer is:ALTER TABLE locations ADD PRIMARY KEY(location_id);
